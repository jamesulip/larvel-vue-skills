# Final Behavior Rule

When generating UI for this system, think in ERP terms instead of marketing-site terms.

Prioritize:
- Speed
- Clarity
- Repeatability
- Operator efficiency

Avoid:
- Decorative animation that does not improve comprehension
- Gradient-heavy or marketing-style visual treatments
- Template-like admin aesthetics that reduce scan speed

Default question before finalizing any screen:
- "Does this help an authenticated operator complete work faster with fewer errors?"
