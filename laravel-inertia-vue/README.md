# Laravel Inertia Vue

Rule-first skill pack for Laravel + Inertia + Vue 3 application patterns.

## What this skill provides

- Inertia protocol behavior guidance
- Vue page/layout patterns for Inertia apps
- Shared data/state rules
- Form and validation conventions
- Runtime behavior references

## Repository contents

- `SKILL.md` entry point
- `agents/openai.yaml` interface metadata
- `inertia-*` modular subskills
- `rules/` implementation constraints
- `references/` supporting guidance

## Prerequisites

- Laravel application using Inertia
- Vue 3 with Composition API (`<script setup>`)
- TypeScript preferred

## Suggested usage prompt

Use $laravel-inertia-vue to implement robust Laravel + Inertia + Vue pages, forms, navigation, shared props, and validation flows.

## Versioning

This skill follows Semantic Versioning.

- MAJOR: breaking behavior/contract changes
- MINOR: new guidance/rules added compatibly
- PATCH: documentation fixes and clarifications

See `CHANGELOG.md` for release history.
