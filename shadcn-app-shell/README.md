# Shadcn App Shell

Skill pack for building a persistent Laravel + Inertia + Vue internal operations shell using shadcn-vue conventions.

## What this skill provides

- App shell architecture for authenticated dashboards
- Shared page-template UI/UX consistency contract
- Workspace navigation and persistent context patterns
- Standardized list/table behavior for index pages
- CRUD form conventions with Inertia behavior alignment
- System feedback and production workflow guidance

## Repository contents

- `SKILL.md` entry point
- `agents/openai.yaml` interface metadata
- behavior standards in top-level markdown files

## Prerequisites

- Laravel + Inertia + Vue setup available
- shadcn-vue initialized in the target app
- Vue 3 Composition API (`<script setup>`)

## Suggested usage prompt

Use $shadcn-app-shell to scaffold an authenticated Laravel + Inertia + Vue application shell optimized for internal operations workflows.

## Versioning

This skill follows Semantic Versioning.

- MAJOR: breaking architecture or behavior changes
- MINOR: new standards and modules
- PATCH: clarifications and non-breaking corrections
