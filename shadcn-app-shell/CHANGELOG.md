# Changelog

All notable changes to this skill are documented in this file.

## [1.0.1] - 2026-03-02

### Added

- New `page-template-ui-consistency.md` standard to enforce consistent UI/UX anatomy across generated templates/pages.

### Changed

- `SKILL.md` now requires applying the page-template consistency standard before generating template/page output.
- Starter, table, form, and final behavior standards now reference one shared page anatomy contract.

## [1.0.0] - 2026-03-02

### Added

- Initial public release of `shadcn-app-shell`
- Standards for app shell layout, navigation, data tables, forms, and feedback
- Production workflow and final behavior guidance
